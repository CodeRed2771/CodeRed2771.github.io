Welcome to Code Red's Complete Guide to the FRC Controls System. We have articles on a variety of topics including programming, wiring, hardware components, and more.

Making a complete guide to the FRC controls system is a lot of work, so we are asking the visitors to our site to assist us in adding content. Any time you can spare to this project would be helpful, even if you only submit half of a page. All you need to do to submit a page is shoot an email to coderedcontrols@gmail.com with the page title and content. It does not need to be HTML formatted. We can do that. The hard part is actually writing all of the content, which you are doing! If you find a typo or other error in a pre-existing page, we encourage you to send an edited version to us so we can update it as quickly as possible. Also, If you include an author name (nicknames encouraged!), we will happily give you credit. Thank you for your support of our project.

Finally, we have organized the programming content into a lesson format. It covers the same content as the normal articles, but introduces it in a paced format and includes relavant information in the same lesson. It's worth taking a look at. Programming Lessons
